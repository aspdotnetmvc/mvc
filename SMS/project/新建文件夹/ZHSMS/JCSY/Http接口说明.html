http接口说明

调用URL地址http://59.36.98.104:3080/smshttp 字符编码GBK 提交方式 GEG/POST
  • act=getbalance   查询用户短信量

    • 参数名称

    参数格式

    参数说明

    unitid

    Int

    企业代码

    username

    String

    用户帐号

    passwd

    String

    用户密码

    返回值

    提交成功返回余额，失败见下面的错误代码详解

    示例

    http://59.36.98.104:3080/smshttp?act=getbalance&unitid=168&passwd=123456&username=test
  • act=uppasswd   更改用户密码

    • 参数名称

    参数格式

    参数说明

    unitid

    Int

    企业代码

    username

    String

    用户账号

    passwd

    String

    用户密码

    newpasswd

    String

    返回值

  • act=sendmsg     发送短信
    		•

    参数名称

    参数格式

    参数说明

    unitid

    Int

    企业代码

    username

    String

    用户帐号

    passwd

    String

    用户密码

    msg

    String

    发送的短信内容，发送手机70字，发送小灵通50字

    phone

    String

    发送号码（号码以’,’号分隔）

    port

    String

    扩展端口号

    sendtime

    String

    发送时间( 格式1900-01-01 00:00:00)

    返回值

    状态,批号,说明

    示例
    http://59.36.98.104:3080/smshttp?act=sendmsg&unitid=100&username=test&passwd=test&msg=测试&phone=13911111111,18911111111&port=&sendtime=2008-01-01 12:00:00
  • act=getstatue 取回执
    		•

    参数名称

    参数格式

    参数说明

    unitid

    Int

    企业代码

    username

    String

    用户帐号

    passwd

    String

    用户密码

    rowid

    int

    获取方式(0 正常接收.1重新接收当天数据.)

    返回值

    格式: 批号|^|号码|^|状态

    两条记录这间用|;|分隔

    316|^|15800000000|^|DELIVRD|;|317|^|13300000000|^|DELIVRD|;|318|^|13900000000|^|DELIVRD|;|319|^|18800000000|^|'UNDELIV

     

    示例
    http://59.36.98.104:3080/smshttp?act=getstatue&unitid=100&username=test&passwd=test&rowid=0
  • act=smsrecord 获取回复信息
    		•

    参数名称

    参数格式

    参数说明

    unitid

    Int

    企业代码

    username

    String

    用户帐号

    passwd

    String

    用户密码

    rowid

    int

    获取方式(0 正常接收.1重新接收当天数据.)

    返回值

    格式: 序号|^|上行号码|^|内容|^|时间|^| sp端口号

    两条记录这间用|;|分隔

    32221|15800000000|^|这是回复测试|^|2000-07-16 15:10:27|^|1065788|;|32222|^|15800000000|^|这是回复测试2|^|2000-07-16 15:10:27|^|1065788

    示例
    http://59.36.98.104:3080/smshttp?act=smsrecord&unitid=100&username=test&passwd=test&rowid=0

    参数说明
